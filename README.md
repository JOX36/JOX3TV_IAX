# JOX3 TV X

Reproductor IPTV para Android TV y dispositivos móviles con interfaz futurista estilo neon.

## Características

- 📺 Soporte Xtream Codes API y listas M3U
- 🎬 Reproductor con ExoPlayer (HLS, DASH, MP4)
- 📋 Multi-cuenta con cambio rápido
- ⭐ Favoritos y continuar viendo
- 🔍 Búsqueda en tiempo real
- 📡 EPG en vivo (guía de programación)
- 🖼️ Carrusel de recomendaciones
- 🔒 Bloqueo de pantalla
- 📌 Picture-in-Picture (PiP)
- 🎨 Interfaz neon/cyberpunk

## Requisitos

- Android 8.0+ (API 26)
- Dispositivo Android TV o móvil

## Construir

```bash
./gradlew assembleDebug
```
